# kindred-legal

Public mirror of the [Kindred app](https://github.com/mr00jay1-lab/Kindred)'s legal documents, served via GitHub Pages.

| Document | Live URL |
|---|---|
| Privacy Policy | <https://mr00jay1-lab.github.io/kindred-legal/privacy/> |
| Terms of Service | <https://mr00jay1-lab.github.io/kindred-legal/terms/> |

These URLs are submitted to App Store Connect and Google Play Console as the canonical privacy and terms references for the Kindred app.

## Updating

The source of truth lives in the [main Kindred repo](https://github.com/mr00jay1-lab/Kindred) under `legal/`. After editing there, copy the updated files into `docs/privacy.md` and `docs/terms.md` here and push — GitHub Pages rebuilds within ~1 minute.

## Notes

- These are **starter drafts** generated for phase 4 development. They cover App Store / Play Store / UK GDPR / UK consumer-law basics so the app can publish a privacy and terms URL for store review.
- A solicitor review is scheduled before phase 10 public release — see [`tasks/todo.md`](https://github.com/mr00jay1-lab/Kindred/blob/Dev/tasks/todo.md) row "Solicitor review booked before phase 10 public release".
